<h1> Amazon Scraper </h1> 

<h3> Node JS  </h3> 

<p>  • This application works in accordance with Twilio, which is a message service company. </p>

<p>  • User can put in the name and URL of the target product, along with the target price. </p>

<p>  • Whenever the condition of price is fulfilled, the user recieves a message on their mobile phones.  </p>

<p>  • This project makes used of axios and cheerio to get data from amazon URL and to filter data on basis of HTML attributes. </p>
